Big Data and Visualisation Projects

Title: Analysing religious conflicts through the lens of global media

Purpose: Final project for COMP4651 - Cloud Computing at HKUST

Instructor: Prof. Wei Wang

Duration: February 2017 - May 2017

Team Members: Shivang G, Hauton T, Prashanth C, Johnny W

Result: Overall course Grade A

Project Brief
To use parallel processing in the cloud to manipulate large sets of data and produce useful results.

Project Overview
We processed the Global Database of Events, Languages and Tones (GDELT) to summarise news reports of violent religious conflicts from around the world and highlighted trends that we noticed through visualisations projected on a world map.

Title: Visualising international relations through the lens of global media

Purpose: Final project for COMP4901F - Data Visualisation at HKUST

Instructor: Prof. Huamin Qu

Duration: February 2017 - May 2017

Team Members: Shivang G, Suyoung P, Kang EJ, Prashanth C

Result: Overall course Grade A

Project Brief
To create a dynamic, large-scale data visualisation using the D3.js library

Project Overview
Using Google BigQuery, the GDELT database was analysed for articles from around the world that spoke about relations between two countries. A relative strength was determined for these relations and then visualised on a world map.
